Application de Gestion de Scolarité

Ce projet est une application en ligne de commande développée pour gérer de manière complète la scolarité des étudiants en BUT.

Fonctionnalités
Interpréteur interactif : Un système de commandes permettant de consulter et gérer le parcours académique complet (inscriptions, saisie des notes, suivi du cursus).
Logique métier complexe : Automatisation des décisions de jury et calcul des moyennes compensées (RCUE).
Suivi des statuts : Gestion précise des situations académiques (admissions, ajournements, gestion des dettes).

Langage : C
